# Library Management System
A simple Library Management System developed using C++ and Singly Linked List.
## Features
- Add Book Details
- Display Book Details
- Add Membership Details
- Display Membership Details
- Add Buyer Details
- Display Buyer Details
- Issue Books
- Display Issued Book Details
- Search Book by Book ID
- Search Buyer by Mobile Number
- Search Issued Book Details
- Dynamic Memory Management using Linked List
## Concepts Used
- Structures
- Pointers
- Singly Linked List
- Dynamic Memory Allocation
- Searching Operations
