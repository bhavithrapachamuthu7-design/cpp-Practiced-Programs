# Employee and Employer Management System
A simple Employee and Employer Management System developed using Singly Linked List.
## Features
* Add Employer Details
* Add Multiple Employee Details
* Display Employer Information
* Display All Employee Records
* Display Employee Positions
* Find Employee with Highest Salary
* Calculate Total Salary (Salary + Bonus)
* Dynamic Memory Management
## Concepts Used
* Structures
* Pointers
* Singly Linked List
* Dynamic Memory Allocation
* Functions
* Searching and Traversal
