# Hospital Management System
A Hospital Management System developed with Priority Queue and Stack concepts to manage patient treatment based on priority levels.
## Features
* Add Patient Details
* Assign Priority Levels (High, Medium, Low)
* Automatic Priority-Based Patient Queue
* Give Treatment to Highest Priority Patient
* Search Patient by Patient ID
* Display Waiting Patients
* Count Total Waiting Patients
* Display Patients by Priority
* Store Treatment History
* View Last Treated Patient
## Data Structures Used
### Priority Queue (Linked List)
* Manages waiting patients based on priority.
* High-priority patients are treated first.
### Stack (Linked List)
* Stores treatment history.
* Last treated patient is displayed first (LIFO principle).
## Concepts Used
* Classes and Objects
* Constructors
* Linked Lists
* Priority Queue
* Stack
* Dynamic Memory Allocation
* Searching
* Data Management
